Welcome to Home Test!

Requirements:
Before running any python file please run "requirements.sh".
This will install all required modules for this project to work.
The modules that will be installed are the following:
- pika
- pandas
- plotly

Instructions:
Run with python in the following order:
1. data_proccessor.py
2. graph_generator.py
3. data_producer.py

You can find the project on GitHub:
https://github.com/MirabelleHe/HomeTest
